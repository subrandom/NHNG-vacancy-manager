This is the module for importing vacancy data.

Once you begin, please do not close your web browser or exit the module until the entire process is complete

The data import process goes like this:

  1. Choose the data file and upload it to the server
  2. Add positions in the data file that are not in our database
  3. Remove positions from our database that the data file indicates are filled
    Notes:
  • There are no confirmations or prompts.
  • New items are automatically imported.
  • Positions that no longer exist in the import file are automatically deleted.
  • Held positions that are not marked as enlisted are flagged as invalid and highlighted in red in the position manager.

When you are ready to begin, click the Start button below, or abort by choosing another destination from the menu.
